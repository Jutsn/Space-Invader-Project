# Space Invaders Project
This project is my interpretation of the famous Space Invaders Game. It's a prototype with one level.  
It was my first solo project after completing the Unity-Junior-Programmer-Course and is written in C#.

## Features
- player controller
- health system
- scrolling background
- power-ups
- enemy spawn system
- enemy behaviour
- boss behaviour

## Installation
1. Clone the repository
2. Open the project in Unity 6
3. Press Play
