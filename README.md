# Space Invader Project
My Interpretation of the famous Space Invaders Game. It's a prototype with one level.
It was my first solo project after completing the Unity-Junior-Programmer-Course and is written in C#.

## Content
- player controller
- scrolling background
- power-ups
- enemy behaviour
- boss fight
